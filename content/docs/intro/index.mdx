# はじめに

このサイトには、授業を受けるうえで知っておいてほしい**共通情報**がまとめられています。

授業を始める前に、以下の3つのページをしっかり確認しておきましょう。

## 確認しておくこと

- **[受講ルール](../course-rules)**
  授業への参加ルール、出席、小テストについて説明しています。

- **[生成AIの利用](../ai-usage)**
  授業中と試験中に、生成AIをどのように使ってよいかを説明しています。

- **[成績評価](../grading)**
  成績の内訳（中間試験・定期試験）と評価の仕組みを確認できます。

---

わからないことがあれば、遠慮なく担当教員に質問してください。
