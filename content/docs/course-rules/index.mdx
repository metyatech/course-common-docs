# 受講ルール

## 授業への参加について

:::caution
**授業中に行う小テストに取り組んでいない場合、「欠席」になります。**
:::

- 授業中の演習は、一緒に手を動かしながら学ぶ大切な時間です。
- 分からないことがあれば、遠慮なく質問してください。

## 出席について

:::caution
**5回以上欠席すると、成績がE判定になります。**
詳しくは学生便覧を確認してください。
:::

## 小テストについて

:::important
**小テストは授業の最初に毎回行います。**
:::

- 成績には含まれませんが、自分の理解度を確認するために活用してください。
- 講義のない日（中間テスト・期末テストなど）は実施しません。

## 自主勉強のコツ

:::info
プログラミングは「慣れ」が大事です！
:::

- 授業中に終わらなかった演習問題を、あとでやり直してみる。
- 自分で問題を考えて解いてみる。
- 作りたいものを自分で作る挑戦をしてみる。

分からないことは調べながら進めて大丈夫です。
「調べて理解して使えること」を目標にしましょう。
