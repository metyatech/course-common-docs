# 受講ルール

## 授業への参加について

:::caution
学生便覧にも記載がある通り、

- **オンラインでの環境不備による参加不能**
- **授業中の、問いかけに反応がない、提出物の提出がない、演習に参加していない、小テストを解いていない**

などの場合は、「**欠席**」とすることがあります
:::

## 小テストについて

:::important
**小テストを、基本的に、講義のあった回の次の回の授業の、最初に行います**
:::

- 成績には含まれませんが、自分の理解度を確認するために活用してください。
- 講義のない回（中間テスト・期末テストなど）の次の回は実施しません。
- 例外的に、行わない場合、授業の途中で行う場合もあるかもしれません。

## 自主勉強のコツ

:::note
プログラミングは「慣れ」が大事です！  
とにかくたくさん手を動かして、たくさんコードを書いてみましょう。
:::

たくさん手を動かすには…

- 授業中に終わらなかった演習問題を、あとで解き直してみる。
- 自分で問題を考えて解いてみる。
- 作りたいものを自分で作る挑戦をしてみる。

分からないことは調べながら進めて大丈夫です。
「調べて理解して使えること」を目標にしましょう。
