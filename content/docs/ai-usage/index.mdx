# 生成AIの利用

「生成AIは使っていいの？」と迷ったら、まずは次の3つだけ覚えてください。

:::important
- **生成AIは原則使ってOKです。**
- **そのままコピペして提出するのはNGです。**
- **最後は自分で説明できる状態にしてください。**
:::

この授業では、授業中も試験中も、生成AIを学習や作業の道具として使ってかまいません。

大切なのは、「使ったかどうか」ではなく、「理解しながら使えているか」です。

担当教員から別の指示があった場合は、その指示を優先してください。

## 授業中はこう使えばOK

- 分からない言葉やエラーの意味を聞く。
- 考え方をやさしい言葉で説明してもらう。
- 別の書き方や別解を比べる。
- コードや文章の下書きを作って、そこから自分で直す。

まず自分で少し考えてから使うと、理解しやすくなります。

## 試験中はここに注意

:::info
**中間試験・定期試験でも、担当教員から別の指示がない限り、検索と生成AIの利用を認めます。**
:::

- 生成AIに全部やらせるのではなく、自分の判断で使ってください。
- 提出する答案や成果物は、自分が意味を説明できるものにしてください。
- AIの答えが正しそうでも、そのまま信じずに確認してください。
- 試験ごとに追加ルールが出た場合は、そのルールを優先します。

## これはNGです

:::caution
**次の使い方は認めません。**
:::

- 内容を理解していないものを、そのまま提出する。
- AIが作った答えを、自分で確認せずに提出する。
- 他の学生の解答や成果物を共有したり、コピーさせたりする。
- 個人情報、学校アカウント情報、パスワード、未公開データを入力する。
- 担当教員が禁止したツールや機能を使う。

## 迷ったときは、この3つで判断してください

1. この内容を、自分の言葉で説明できますか。
2. この答えが間違っていたとき、自分で直せますか。
3. 入力してはいけない情報を入れていませんか。

## うまく使うコツ

- 質問は、目的・条件・困っている点を具体的に書く。
- AIの答えが出たら、実行する、読み直す、比べる、の3つで確認する。
- うまくいかないときは、AIとのやり取りも含めて担当教員に相談してください。
