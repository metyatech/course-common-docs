# 生成AIの利用

この授業では、授業中も試験中も、生成AIを学習や作業の道具として使ってかまいません。

大切なのは、「使ったかどうか」ではなく、「理解しながら使えているか」です。

## 基本方針

:::important
**生成AIは原則利用可です。**
ただし、担当教員から別の指示があった場合は、その指示を優先します。
:::

- 調べる、整理する、例を作る、コードや文章の下書きを作る、といった使い方を認めます。
- 生成AIの出力は、必ず自分で読み直し、動作や内容を確認してください。
- 最終的に提出する内容は、自分の言葉で意味を説明できるものにしてください。

## 授業中の利用

- エラーの原因調査、考え方の整理、別解の比較などに使ってかまいません。
- まず自分で考えてから質問すると、理解しやすくなります。
- 分からないままコピペして終わりにしないようにしましょう。

## 試験での利用

:::info
**中間試験・定期試験でも、担当教員から別の指示がない限り、検索と生成AIの利用を認めます。**
:::

- 生成AIに解答を丸投げするのではなく、自分の判断で使ってください。
- 提出する答案や成果物の内容は、自分が説明できることを前提にしてください。
- 試験ごとに追加ルールが出た場合は、そのルールがこのページより優先されます。

## 認めない使い方

:::caution
**次の使い方は認めません。**
:::

- 内容を理解していないものを、そのまま提出する。
- 他の学生の解答や成果物を共有したり、コピーさせたりする。
- 個人情報、学校アカウント情報、パスワード、未公開データを入力する。
- 担当教員が禁止したツールや機能を使う。

## うまく使うためのコツ

- 質問は、目的・条件・困っている点を具体的に書く。
- AIの答えが正しそうに見えても、必ず実行する、読み直す、比べる、の3つで確認する。
- うまくいかないときは、AIとのやり取りも含めて担当教員に相談してください。
