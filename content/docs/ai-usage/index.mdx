# 生成AIの利用

「生成AIは使っていいの？」と迷ったら、まずは次の3つだけ覚えてください。

:::important
- **生成AIは原則使ってOKです。**
- **意味が分からないまま、そのまま提出するのはNGです。**
- **最後は自分で説明できる状態にしてください。**
:::

この授業では、授業中も試験中も、生成AIを学習や作業の道具として使ってかまいません。

担当教員から別の指示があった場合は、その指示を優先してください。

## 使ってよい使い方

### 1. 分からないことを理解するために使う

たとえば、次のような使い方です。

- 「このエラー文の意味を、やさしい言葉で説明して」
- 「このコードが何をしているか、1行ずつ説明して」
- 「この文章を、高校生にも分かる言い方に直して」

### 2. 下書きを作ってもらい、自分で直す

たとえば、次のような使い方です。

- 「レポートの見出し案を3つ出して」
- 「このコードのたたき台を作って」
- 「別の書き方の例を2つ見せて」

この場合も、そのまま出すのではなく、自分で読み直して直してください。

### 3. 比べて学ぶ

たとえば、次のような使い方です。

- 「この書き方と別解の違いを教えて」
- 「この説明のどこが分かりにくいか教えて」
- 「このコードの改善点を3つ挙げて」

## 使ってはいけない使い方

### 1. 意味が分からないまま提出する

これはNGです。

- AIが作ったコードを、自分で読まずにそのまま提出する。
- AIが作った文章を、内容を確認せずにそのまま提出する。

### 2. AIの答えを確認せずに信じる

これはNGです。

- 実行していないコードを、そのまま正しいと思って提出する。
- AIが書いた説明を、教科書や授業内容と比べずにそのまま使う。

### 3. 入力してはいけない情報を入れる

これはNGです。

- 学校アカウントの情報やパスワードを入力する。
- 未公開の成績情報や個人情報を入力する。

## 試験中の使い方

:::info
**中間試験・定期試験でも、担当教員から別の指示がない限り、検索と生成AIの利用を認めます。**
:::

試験中も考え方は同じです。  
「使うこと」自体は問題ではなく、「自分で理解して扱えているか」が大事です。

たとえば、次のような使い方はOKです。

- 「このコードのエラー候補を3つ挙げて」
- 「この説明で見落としている点がないか確認して」

逆に、次のような使い方はNGです。

- 「この問題の答えをそのまま全部作って」
- AIが出した答えを、自分で確認せずにそのまま答案に書く。

## 迷ったときは、この3つで判断してください

1. この内容を、自分の言葉で説明できますか。
2. この答えが間違っていたとき、自分で直せますか。
3. 入力してはいけない情報を入れていませんか。
