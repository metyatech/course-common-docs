# 成績評価

## 成績の構成

| 項目 | 配点 |
|------|------|
| **中間試験** | **20点満点** |
| **定期試験** | **80点満点** |

- 出席に問題がなければ、**中間試験の点数と定期試験の点数の合計が評点**になります。

## 試験について

- 試験は、授業スライドの確認・過去に作ったプログラムのコピペ・検索・生成AIの使用が可の予定です。
- 詳細は各試験の前に担当教員から案内します。

## 出席と成績の関係

:::caution
**出席に問題がある場合、試験の点数に関わらず成績に影響します。**
詳しくは学生便覧を確認してください。
:::

- 5回以上欠席すると、E判定（単位なし）が確定します。
- 授業への参加が成績の土台になります。
