# 成績評価

## 成績の構成

| 項目 | 配点 |
|------|------|
| **中間試験** | **20点満点** |
| **定期試験** | **80点満点** |

- 出席に問題がなければ、**中間試験の点数と定期試験の点数の合計が評点**になります。

## 試験について

- 試験は、授業スライドの確認・過去に作ったプログラムの再利用・検索・生成AIの使用が可です。
- 生成AIの具体的な適正な使い方は、**[生成AIの適正な利用方法について](../ai-usage)** を確認してください。

## 出席について

:::caution
**5回以上欠席すると、成績がE判定になります。**
:::

- 詳しくは学生便覧を確認してください。
